# HASpoolManager

3D Printing Filament Lifecycle Manager for Bambu Lab printers.

## Configuration

- **log_level**: Debug level (debug, info, warning, error)
- **anthropic_api_key**: Optional API key for AI features (label scanning, order parsing)

## Data

All data is stored in `/config/haspoolmanager.db` and is included in HA backups automatically.
