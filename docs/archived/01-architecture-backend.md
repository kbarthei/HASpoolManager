Matching Algorithm\n\nThe match endpoint implements a two-tier strategy:\n\n### Tier 1: Exact match (deterministic)\n\n```typescript\nasync function exactMatch(req: MatchRequest): Promise<Spool | null> {\n  // 1. RFID tag match -- highest confidence\n  if (req.tag_uid && req.tag_uid !== \"0000000000000000\") {\n    const mapping = await db.tagMappings.findByTagUid(req.tag_uid);\n    if (mapping) return { spool: mapping.spool, confidence: 1.0, method: 'rfid_exact' };\n  }\n\n  // 2. Bambu filament index + AMS slot match\n  //    If a spool is already assigned to this slot, and its filament's bambu_idx matches, trust it\n  if (req.printer_id && req.ams_index !== undefined && req.tray_index !== undefined) {\n    const slot = await db.amsSlots.findByPosition(req.printer_id, req.ams_index, req.tray_index);\n    if (slot?.spool_id) {\n      const spool = await db.spools.findById(slot.spool_id);\n      if (spool?.filament.bambu_idx === req.tray_info_idx) {\n        return { spool, confidence: 0.95, method: 'bambu_idx_exact' };\n      }\n    }\n  }\n\n  return null;\n}\n```\n\n### Tier 2: Fuzzy match (scored)\n\nWhen no exact match exists, score all active spools:\n\n```typescript\ninterface MatchWeights {\n  bambu_idx:    number;  // default: 40 -- filament product code match\n  material:     number;  // default: 20 -- PLA vs PETG etc.\n  color:        number;  // default: 25 -- color hex distance (CIE Delta-E)\n  vendor:       number;  // default: 10 -- vendor keyword match\n  location:     number;  // default:  5 -- spool.location == \"ams\" scores higher\n}\n\nfunction fuzzyScore(spool: Spool, req: MatchRequest, weights: MatchWeights): number {\n  let score = 0;\n\n  // Bambu filament index partial match (same prefix = same vendor line)\n  if (req.tray_info_idx && spool.filament.bambu_idx) {\n    if (spool.filament.bambu_idx === req.tray_info_idx) {\n      score += weights.bambu_idx;          // exact product match\n    } else if (spool.filament.bambu_idx.slice(0, 3) === req.tray_info_idx.slice(0, 3)) {\n      score += weights.bambu_idx * 0.3;    // same product line (e.g. GFA = Bambu PLA family)\n    }\n  }\n\n  // Material type match\n  if (req.tray_type && spool.filament.material.toUpperCase() === req.tray_type.toUpperCase()) {\n    score += weights.material;\n  }\n\n  // Color distance (CIE76 Delta-E in LAB space)\n  if (req.tray_color && spool.filament.color_hex) {\n    const reqHex = req.tray_color.slice(0, 6);  // strip alpha\n    const deltaE = cieDeltaE(hexToLab(reqHex), hexToLab(spool.filament.color_hex));\n    // deltaE < 2.3 = imperceptible, < 5 = close, > 20 = totally different\n    if (deltaE < 2.3) score += weights.color;\n    else if (deltaE < 5) score += weights.color * 0.8;\n    else if (deltaE < 10) score += weights.color * 0.4;\n    else if (deltaE < 20) score += weights.color * 0.1;\n    // > 20 = no points\n  }\n\n  // Vendor keyword match (sub_brands or tray_info_idx prefix)\n  if (req.tray_sub_brands) {\n    const vendorName = spool.filament.vendor.name.toLowerCase();\n    if (req.tray_sub_brands.toLowerCase().includes(vendorName)) {\n      score += weights.vendor;\n    }\n  }\n\n  // Location bonus: spools marked as \"ams\" location are more likely in play\n  if (spool.location === 'ams') {\n    score += weights.location;\n  }\n\n  // Normalize to 0-1\n  const maxScore = weights.bambu_idx + weights.material + weights.color\n                 + weights.vendor + weights.location;\n  return score / maxScore;\n}\n```\n\n### Match orchestration\n\n```typescript\nexport async function POST(req: NextRequest) {\n  const body: MatchRequest = await req.json();\n\n  // Tier 1: deterministic\n  const exact = await exactMatch(body);\n  if (exact && exact.confidence >= 0.95) {\n    return Response.json({ match: exact, candidates: [] });\n  }\n\n  // Tier 2: fuzzy\n  const activeSpools = await db.spools.findByStatus('active');\n  const weights = await db.settings.getMatchWeights(); // user-configurable\n\n  const scored = activeSpools\n    .map(spool => ({\n      spool,\n      confidence: fuzzyScore(spool, body, weights),\n      match_reasons: buildReasons(spool, body),\n    }))\n    .filter(s => s.confidence > 0.2)\n    .sort((a, b) => b.confidence - a.confidence);\n\n  const best = scored[0] ?? null;\n  const candidates = scored.slice(1, 6);\n\n  return Response.json({\n    match: best ? {\n      spool_id: best.spool.id,\n      filament_name: best.spool.filament.name,\n      vendor_name: best.spool.filament.vendor.name,\n      confidence: best.confidence,\n      match_method: 'fuzzy',\n      remaining_weight: best.spool.remaining_weight,\n    } : null,\n    candidates,\n  });\n}\n```\n\n---\n\n## 5.
